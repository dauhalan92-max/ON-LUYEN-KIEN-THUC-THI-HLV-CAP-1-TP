# Học viện Huấn luyện viên 2026

Bản này gồm 6 game HTML và cổng chính `index.html`.

## Cách đưa lên GitHub Pages
1. Giải nén file ZIP.
2. Upload toàn bộ nội dung bên trong thư mục lên repository GitHub.
3. Vào Settings → Pages.
4. Chọn Deploy from branch → main → /root.
5. Mở link GitHub Pages được cấp.

## Cài như app trên điện thoại
- iPhone: mở link bằng Safari → Chia sẻ → Thêm vào Màn hình chính.
- Android: mở link bằng Chrome → dấu ba chấm → Thêm vào màn hình chính hoặc Cài đặt ứng dụng.

## Ghi chú bản sửa
Trong từng game, nút `Trang chủ`/`Cổng chính` đã được sửa để quay về màn hình chính tổng hợp, không quay về màn hình đầu của từng game nữa.
